# UI screens

Static, high-fidelity frontend screens for design reference and handoff.

| Screen | Path | Open locally |
|--------|------|----------------|
| Zomato AI — Home / Discovery | `zomato-ai-home/index.html` | Double-click or `python -m http.server 5500` from `screens/` then visit `/zomato-ai-home/` |

These are **standalone** (no Phase 3 API required). Wire to the API by porting markup into `Phase4/public` or a Next.js app.
